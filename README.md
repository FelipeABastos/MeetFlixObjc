# Meetflix

This is a project in Objective C for training, totally based at Netflix layout.

## Requirements

```
iOS 13+
Xcode 11
Objective C
```

## This class uses:

```
UITableView
UICollectionView
UIProgressView
UIStackView
Effect View with blur
```
